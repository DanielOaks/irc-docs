Topic: Gline list addition: Drones

Date of conclusion : 12 Sep 2003
Result: PASSED (yes: 19, no: 2, abstain: 3)
Description:

The following vote has closed:
                                                                                
Gline list addition: Drones
                                                                                
This vote is to allow the Glining of drones.
                                                                                
For the purpose of this Gline reason/vote, a client is
considered a drone if it meets all of the following
requirements:
                                                                                
- IRC connection established from a system without the
knowledge of a user of the system.
- IRC connection established by a piece of trojan software
with the primary purpose of flooding and/or participating
in denial of service attacks.
                                                                                
Therefore, this vote does not cover *every* illegitimate
client connection established from a compromised system
(hackers on bnc's, xdcc bots whose primary purpose is to
distribute files, possibly others).  Some admins would like
to see such clients Glined, and some would like to see them
dealt with in other ways.  Thus, they should be handled
under another list item/vote which any admin should feel
free to call if so desired.
